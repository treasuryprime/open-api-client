## Create Account Application
<EndpointMethod method="post" route="/apply/account_application" />
<SplitSection>
  <Left>
    To create an application for a new bank account, create an Account Application object.

    This example demonstrates the creation of a Personal account. Business accounts are very similar, but will utilize a few different fields. Please see our **account opening guide** for more information.

    ##### Parameters
    ***
    <Parameter name="account_id" type="string" filterable required>Bank account number.</Parameter>
    ***
    <Parameter name="account_product_id" type="string">Either checking or savings. Non-DDA accounts are currently not supported.</Parameter>
    ***
    <Parameter name="address" type="string">Address on file associated with the bank account.</Parameter>
    ***
    <Parameter name="available_balance" type="string">Available funds in this account. If not available, the value will be null.</Parameter>
    ***
    <Parameter name="business_ids" type="array[string]">Array of IDs of Business objects representing all businesses that own this account. Always empty for personal accounts.</Parameter>
  </Left>
  <Right>
    <Panel title="Post Request">
      ```
      curl https://api.treasuryprime.com/apply/account_application \
        -u $API_KEY_ID:$API_KEY_VALUE
        -H 'Content-Type: application/json' \
        -d '{
              "deposit_id": "adpt_01d5w6xx0tvs",
              "person_applications": [
                {
                  "id": "apsn_01d5w6yaa6vt",
                  "roles": ["owner", "signer"]
                }
              ],
              "primary_person_application_id": "apsn_01d5w6yaa6vt",
              "account_product_id": "apt_11gqk87qmrax"
      ```
